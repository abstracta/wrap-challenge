This testcases are running in Ruby and Cucumber, so, follow the instructions to run the tests.

1 - Install Ruby (Ruby is sopported in Windows, Linux and Mac. In Linux I reccomend to install via RVM)
2 - Install DevKit if you are using Windows
3 - In your console, execute this command : "gem install bundler"
4 - Now, execute : "bundle install"
5 - To execute the testcase and make a report, execute : "cucumber -t @test_case --format html --out output/report.html --format pretty"
